# Ater Recipe


| Ingredient | Weight | Volume  |
| ---------- | ------ | ------- |
| Sekar      |        | 2 cups  |
| May        |        | 1 cup   |
| Hamda      |        | nos     |
| Mazaher    |        | chwayit |
| Mawared    |        | chwayit |


### Sugar to Water Ratio

- Ater 2:1
- Simple Syrup 1:1

### Prep

1. Ghliyoun 3a <u>nar khafife/wasat</u> ma3 1/2 hamda la ma yeskaro
2. Awwal ma yeghlo w ysiro bl sameke l baddak yeha (kl ma tawwalo ghale kl ma asmak) tfi 3anoun w 7ot no2tit mazaher w mawared w trekoun yeberdo
3. 7otoun bi annine bl borrad
